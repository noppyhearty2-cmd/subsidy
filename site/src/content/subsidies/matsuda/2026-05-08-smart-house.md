## こんな人が対象

松田町内の住宅にスマートハウス設備を設置する個人

## いくらもらえる？

| 設備 | 補助額 |
|------|--------|
| 太陽光発電システム（10kW未満） | **1kWあたり2万円（上限10万円）** |
| HEMS | **1万円（定額）** |
| 家庭用燃料電池システム | **5万円（定額）** |
| 潜熱回収型ガス給湯器 | **5万円（定額）** |
| ヒートポンプ式給湯器 | **5万円（定額）** |
| V2H充給電設備 | **導入費の1/2以内（上限5万円）** |
| 定置用リチウムイオン蓄電池 | **5万円（定額）** |

予算額：375万円

## いつまでに申し込む？

- **実績報告：設置完了日から30日以内、または令和9年3月15日（月）のいずれか早い日まで**
- 予算がなくなり次第終了

## 申し込みのポイント

1. **設置後に実績報告が必要**　設置工事完了後30日以内に実績報告書を提出してください。
2. **お問い合わせ：環境上下水道課 電話：0465-83-1227**
