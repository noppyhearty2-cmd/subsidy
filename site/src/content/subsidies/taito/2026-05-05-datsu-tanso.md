## こんな人が対象

**台東区内建物の所有者または所有者の承諾を得た方**が対象です。

- 個人（住宅・マンション）
- 法人・管理組合
- 事業所・店舗の所有者

## いくらもらえる？

| 設備 | 補助額 |
|------|--------|
| **LED照明改修** | 対象経費の **50%**（上限 **75万円**） |
| **太陽光発電システム** | 5万円/kW（上限 **20〜50万円**） |
| **蓄電システム** | 1万円/kWh（上限 **10万円**） |
| **燃料電池（エネファーム）** | **14万円**/台 |
| **窓・外壁断熱改修** | 対象経費の **20%**（上限 **15万円**） |
| **高反射率塗料** | 対象経費の **20%** |
| **省エネ機器** | 対象経費の **50%**（上限 **75万円**） |

## いつまでに申し込む？

- **申請期限**：**2027年2月15日**（月）まで
- **重要**：令和8年度より**事前申込型**に変更。工事前に申し込みが必要です

## 申し込みの3つのポイント

1. **令和8年度から事前申込型に変わった**：以前は事後申請でしたが、令和8年度より工事前の申し込みが必須になりました。工事を始める前に必ず申し込み手続きを行ってください
2. **LEDと省エネ機器は経費の50%と高率**：LED改修・省エネ機器はそれぞれ上限75万円で経費の半額が出ます。事業所の照明一括LED化などに活用できます
3. **東京都補助と組み合わせ可能**：東京都の補助金との重複申請が可能です。都と区の両方に申請することで補助金を最大化できます
